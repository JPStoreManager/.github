## JP Store Manager Application

가게의 관리를 위한 어플리케이션 개발

관리의 대상은 아래와 같다.

1. 매출, 매입, 고정비 등의 재무와 관련된 데이터 관리

2. 재무 데이터를 바탕으로 동년, 월, 일에 대한 비교 및 앞으로의 예측지표 생성을 통한 insight 제공

3. 가게의 일정 관리 (아르바이트, 손님 예약 등)
